# 核心模块深度分析任务

基于项目全貌分析，以下是需要逐一深度分析的核心模块列表。

## 任务列表

- [x] 1. Agent 引擎 - 基于 LangGraph ReAct 模式的主 Agent 创建与运行，含工厂模式组装、特性管理、SSE 流式响应
- [x] 2. 中间件链 - 14 个有序中间件的预处理/后处理管道
- [x] 3. 模型工厂 - 6 个 LLM Provider 的统一创建入口，通过模型名称前缀自动路由
- [x] 4. 沙箱系统 - Local + Docker 双 Provider 的代码执行沙箱
- [x] 5. 工具系统 - 动态工具加载器，支持内置/MCP/社区工具
- [x] 6. 子 Agent 系统 - 双线程池异步执行器 + 注册表
- [x] 7. 记忆系统 - 双层记忆架构：短期文件 + 长期向量化存储
- [x] 8. MCP 集成 - 多服务器 MCP 客户端，支持 stdio/SSE/HTTP
- [x] 9. 技能系统 - SKILL.md 格式的技能发现、解析、验证和加载
- [x] 10. 护栏系统 - 可插拔的授权检查引擎
- [x] 11. 配置系统 - YAML/JSON 配置加载 + 环境变量覆盖
- [x] 12. Gateway API - FastAPI 应用入口与 10 个 RESTful 路由
- [x] 13. IM 渠道桥接 - 飞书/Slack/Telegram 三平台消息桥接
- [x] 14. 上传管理 - 文件存储、元数据管理和文档格式转换
- [x] 15. 反射系统 - 通过字符串路径动态加载模块
- [x] 16. 社区工具 - Tavily/Jina/Firecrawl/DuckDuckGo 集成
